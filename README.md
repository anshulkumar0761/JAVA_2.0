# Learn Java 🚀

Welcome to **Java 2.0**, a beginner-friendly guide and resource hub for mastering the fundamentals of Java programming.  
This repository is designed to help learners build a strong foundation in Java, with practical examples, exercises, and projects.

---

## 📚 What You'll Learn
- Basics of Java syntax and structure
- Variables, data types, and operators
- Control flow (if-else, loops, switch)
- Object-Oriented Programming (OOP) concepts:
  - Classes & Objects
  - Inheritance
  - Polymorphism
  - Encapsulation
- Exception handling
- Collections framework
- File handling
- Introduction to Java APIs

---

## 🛠 Prerequisites
- Basic understanding of programming concepts (optional but helpful)
- Installed [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html)
- A code editor or IDE (e.g., IntelliJ IDEA, Eclipse, or VS Code)
